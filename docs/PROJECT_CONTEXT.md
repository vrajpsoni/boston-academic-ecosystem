# CS617 Final Project

## Topic
Boston: A Hub of Academic Excellence

## Main Goal
Create a modern storytelling data visualization website about how students shape Boston.

The project should focus on:
- universities
- student population
- housing impact
- startups
- research ecosystem
- employment
- innovation culture
- city identity

## Website Style
Reference:
https://avanith12.github.io/Unemployment-in-Massachusetts-vs.-the-U.S/

The website should:
- feel modern
- be visually engaging
- use real datasets
- use storytelling
- use interactive charts/maps
- resemble professional data journalism

DO NOT:
- make cinematic animations
- overcomplicate the UI
- generate fake data

## Technologies
- HTML
- CSS
- JavaScript
- D3.js
- Plotly.js
- CSV datasets
- GitHub Pages

## Important
Before coding:
1. Analyze proposal PDFs
2. Analyze reference projects
3. Suggest strongest charts
4. Suggest best datasets
5. Build data story first
